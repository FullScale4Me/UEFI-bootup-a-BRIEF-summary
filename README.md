UEFI bootup: a BRIEF summary

The UEFI firmware contains its own Boot Manager 
that operates independent of hard disk Boot Order 
used by BIOS aka Legacy booting.

My intent was a VERY brief writeup that even a new to 
Linux user could digest and achieve >90% comprehension 
in ONE read. There are plenty of comprehensive writings 
on this topic to be found on the web, Google has almost all of them :-)
